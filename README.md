## vim-starfyre

A Vim plugin for syntax highlighting Starfyre files.

## Installation

### Using [vim-plug](https://github.com/junegunn/vim-plug)
1. Add `Plug 'sparckles/vim-starfyre'` to your `vimrc` file.
